About
Q: What is CCExtractor? What is CCExtractor Web?

A: CCExtractor is a tool that extracts subtitles from video files. Originally written as a desktop program, CCExtractor can now be run any device with a network connection through CCExtractor Web (you are here!).

Q: How do I get started?

A: Simple! Create an account or login to access the dashboard and upload video files for processing. A full guide for using CCExtractor Web can be found here.
